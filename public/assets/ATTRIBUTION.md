# RoleTop Starter Pack — Attribution & License

All 39 assets in this pack (`public/assets/`) were authored from scratch for
the RoleTop VTT Phase 6 project. They ship under the same license as the
project (MIT). **No third-party attribution is required** to redistribute,
modify, or remix them.

| Category    | Count | Format | Typical size |
| ----------- | ----: | ------ | ------------ |
| Conditions  |    16 | SVG    | ~0.5 KB each |
| Tokens      |    12 | SVG    | ~1–2 KB each |
| Tiles/props |     8 | SVG    | ~1 KB each   |
| Battle maps |     3 | PNG    | 140–660 KB   |

## Why SVGs?

All character, condition, and prop artwork is vector so it stays crisp at any
zoom and weighs a few hundred bytes each. The existing token sprite atlas
(`src/render/assets.ts`) rasterises them on demand.

## Why procedural PNGs for maps?

Battle maps need a raster footprint (stone textures, grass grain, wood
planks) that would cost orders of magnitude more as SVG. The PNGs are
generated by the script below with Pillow; regenerate or tweak at will.

## Regenerating the maps

The generator script is inline in the commit that introduced this pack. To
re-run after a visual tweak, see the `python3 << PYEOF` blocks in the PR
description — they're a single-file, zero-dependency Pillow script.

## Swapping in upstream art packs

If you want to replace or extend these assets with community work
(Forgotten Adventures, 2-Minute Tabletop, Kenney, game-icons.net), use the
**Import manifest URL** field in the in-app asset library — the loader at
`src/features/assetManifest.ts` normalises both common manifest shapes and
resolves relative asset URLs against the manifest's host.

Recommended upstream sources:

- **Kenney.nl** — CC0, no attribution required. Excellent for tiles.
- **game-icons.net** — CC-BY 3.0. 4000+ mono icons ideal for conditions.
- **Wikimedia Commons** — Public Domain / CC0. Historical maps, tavern art.
- **OpenGameArt.org** — mixed; filter by CC0 for redistribution-safe content.

When swapping in CC-BY content, append an entry to this file with the
author, source URL, and license, and make sure the manifest's `attribution`
field names them.
